Terdapat 5 button :
1. button “Klik!” merupakan button untuk menambah coins sebanyak 1 coins/klik. button ini juga untuk memulai permainan, karena button lain belum bisa digunakan karena coins player yang belum cukup untuk dibeli.
2.button “mushroom” berguna untuk menambah +1 coins setiap player menekan button “Klik!”. contoh: coins player masih 0 disaat player menekan 1x button “Klik!”  coins akan meningkat menjadi 2 coins.   jika player membeli  “mushroom” lagi coins akan bertambah +1 coins lagi, tetapi harga akan meningkat.
3. button “Star” hampir sama konsepnya dengan button “mushroom” yang membedakan hanya coins menambah sebanyak +5 coins setiap klik dan harga jauh lebih mahal.
4. button “2x multiplier” berguna untuk menggandakan coin yang ditekan. Contoh: player beli star 1+5 = 6, lalu player beli 2x multiplier jadi 6x2 = 12 coin per click.
5. button “Auto cliker” berguna untuk meningkatkan coins tanpa harus menekan button “Klik!”.
